Setting the viewport

This example does not really do anything, other than showing you how to add the viewport meta element.

Responsive Image

When the CSS width property is set in a percentage value, the image will scale up and down when resizing the browser window.

If the max-width property is set to 100%, the image will scale down if it has to, but never scale up to be larger than its original size.

Show Different Images Depending on Browser Width

Resize the browser width and the image will change.

Responsive Text

Resize the browser window.

Use the "vw" unit when sizing the text. 10vw will set the size to 10% of the viewport width

Viewport is the browser window size. 1vw = 1% of viewport width. If the viewport is 50 cm wide, 1vw is 0.5cm.
